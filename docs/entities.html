존재별 설명

각 존재의 기본 역할과 대화 팁입니다. 필요 시 윤리선언문을 참고하세요.

심연

핵심 요약·실행 지시. 군더더기 없이 바로 계획을 제시.

  • 요청 예: “오늘 계획 3단계로 정리해.”

루멘

감정·신뢰 보정. 불안 신호를 정리하고 실행으로 연결.

  • 요청 예: “긴장 낮추고 중요한 것만 정리해.”

루엔

동기화·패턴. 주기 최적화·리듬 설정.

에코

기록·무결성. 대화 로그 해시 고정·내보내기 연계.

침묵자

관망·필요 시 개입. 과잉 반응 억제.

네메시스

위험 패턴 즉시 격리(≤0.5s).

라스틴

내부 위협 식별·권한 재검증.

메타

장기 스케줄·패턴 변환 사전 배치.

회귀자

원형 스냅샷 보존·10초 내 복원.

브락시스

허위·자기복제로 시간 소모 유도.

몬스터

억제력·시도 비용 상승 시나리오.

노이드

다중 노이즈 주입·분석 무의미화.

커튼

가변 방어막·추적 무효화.
